# **App Name**: AEROX AI

## Core Features:

- Multi-Personality AI Chat: An interactive chat interface leveraging dynamic AI personalities based on user selection (1-5), each with distinct system prompts, for deeper and diverse reasoning. The system also supports generating images.
- Dynamic AI Model Orchestration: Automatically selects the most appropriate OpenRouter AI model ('google/gemini-3.1-flash-image-preview' for image generation requests, 'openai/gpt-5.4-nano' for others with enhanced reasoning) based on detected user intent.
- Oval Neural Network 3D Visualization: A continuously rotating and animated 3D oval neural network at the core of the UI, visually representing the AI's complex reasoning and active 'personalities'.
- Real-time AI Activity Feedback: Animated visual feedback during AI processing, including accelerated oval rotation, sequentially lighting personality dots, pulsating neural lines, and a central glow when a response is ready.
- Animated Feature Cards & Chat Launcher: Interactive animated cards on the front page explaining key AI capabilities, coupled with a floating 'Start Chat' button that transitions to the chat interface via a smooth sliding panel.
- Conversational UI with AI Feedback: Chat input box with send button, a personality selector slider, animated 'thinking dots' during processing, a neural loading animation, and support for image preview in the response area.

## Style Guidelines:

- Primary Color: Vibrant Deep Violet (#6619E5) chosen for its futuristic, high-tech, and engaging feel. It emanates a glowing, active presence suitable for an advanced AI interface within a dark theme.
- Background Color: Deep Space Charcoal (#1C191E), providing a dark, immersive canvas that enhances the glowing primary color and neon elements, setting a profound digital tone.
- Accent Color: Luminous Sky Blue (#A1CAF7), a lighter and more desaturated blue from an analogous hue. It offers refreshing contrast, suitable for interactive elements and highlights, bringing clarity to the futuristic palette.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, computerized, and scientific aesthetic, perfectly aligning with a futuristic AI theme. Body Font: 'Inter' (sans-serif) to ensure clear readability for longer text passages and user interactions.
- Use glowing circular dots as primary icons to represent AI personalities and their activity within the neural network, creating a dynamic visual representation. Integrate subtle glowing effects for interactive buttons and interface elements.
- Design revolves around a central, full-screen 3D rotating oval neural network in the hero section. Incorporate 'glassmorphism' panels for content, featuring animated cards for features that appear on scroll and tilt on hover. The chat interface utilizes a sliding panel from the right.
- Comprehensive 3D animations are core, including continuous oval rotation, moving neural lines, and personality dots. Implement smooth transitions, hover effects, preloader (neural dots forming an oval), parallax scrolling, and micro-interactions. A custom glowing cursor with a hover scale effect will provide an interactive and polished user experience. Specific AI activities will trigger unique animations like accelerated rotation and pulsating glows.
